# Alfred-Colors-workflow

Convert css colors(**hex, rgb, hsl**)

The keyword is **c**, you can change it with you habits.

![gif](c.gif)

## Format
### hex 

* #ffffff
* ffffff
*  #fff
* fff

### rgb 

> rgb(...) is optional

* rgb(255, 255, 255)
* rgb(255|255|255)
* rgb(255 255 255)
* rgb(255-255-255)

### hsl

> hsl(...) is optional

* hsl(0, 0%, 100%)
* same as rgb
