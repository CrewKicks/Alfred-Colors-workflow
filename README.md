# Alfred-Colors-workflow

Css colors conversion(**hex, rgb, hsl**)

The keyword is **c**, you can change it with you habits.

![demo](demo.gif)

# Pre Install

```
brew install ruby
gem install chunky_png
```

# Format

### keywords

* black => #000
* darkblue => #00008b
* yellowgreen => #9acd32
### hex 

* #ffffff
* ffffff
* #fff
* fff

### rgb 

> rgb(...) is optional

* rgb(255, 255, 255)
* rgb(255|255|255)
* rgb(255 255 255)
* rgb(255-255-255)

### hsl

> hsl(...) is optional

* hsl(0, 0%, 100%)
* same as rgb
